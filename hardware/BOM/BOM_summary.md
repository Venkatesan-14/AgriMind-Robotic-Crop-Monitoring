# BOM Summary

The AgriMind prototype is built using commonly available embedded system and robotics components.

## Main Components

| Component | Quantity | Purpose |
|-----------|---------:|---------|
| ESP32 Microcontroller | 1 | Main controller |
| pH Sensor Module | 1 | Soil pH measurement |
| Capacitive Soil Moisture Sensor | 1 | Soil moisture measurement |
| BMP280 Sensor | 1 | Temperature and pressure measurement |
| NEO-6M GPS Module | 1 | Location tracking |
| OLED Display | 1 | Local display |
| L298N Motor Driver | 3 | Motor control |
| BO DC Motors | 4 | Rover movement |
| Linear Actuator Motor | 1 | Soil probe movement |
| Li-ion Battery | 1 | Power source |
| Buck Converter | 1 | Voltage regulation |

For the complete list of components, refer to **AgriMind_BOM.xlsx**.
