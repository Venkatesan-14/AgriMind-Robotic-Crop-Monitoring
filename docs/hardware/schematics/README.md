# Hardware Schematics

This folder contains the system-level schematic of the AgriMind prototype.

The schematic was designed using KiCad and represents the electrical connections between the ESP32 controller, sensors, motor drivers, power supply, and actuator used in the prototype.

## Files

- **AgriMind_System_Schematic.pdf** – Printable version of the schematic.
- **AgriMind_System_Schematic.png** – Preview image for quick viewing.
- **AgriMind_System_Schematic.svg** – Editable vector version (optional).

The schematic reflects the prototype developed during the project and is intended for documentation purposes.
