# Firmware

This folder contains the firmware developed for the AgriMind prototype.

## Features

- Controls rover movement
- Operates the soil sampling probe
- Reads soil pH and moisture sensors
- Acquires GPS coordinates
- Hosts a Wi-Fi dashboard for live monitoring
- Displays soil health recommendations based on sensor readings

## Development Environment

- Arduino IDE
- ESP32 Development Board
- Programming Language: C++
